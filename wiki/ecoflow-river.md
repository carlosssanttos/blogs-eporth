# EcoFlow — Linha River (Estações Compactas)

Estações de energia portáteis da linha de entrada EcoFlow. Ideais para uso pessoal, camping, home office e backup leve. Todas usam bateria LFP com 3.000 ciclos (~10 anos).

> **Preços:** consulte sempre [[precificacao-powersafe]] — fonte única e atualizada.

---

## River 3

### Copy / Descrição
Bateria portátil compacta com tecnologia X-GaNPower, eficiência energética superior, design 30% menor que gerações anteriores e operação silenciosa. Solução versátil para backup doméstico e uso outdoor.

### Especificações Técnicas
| Spec | Valor |
|------|-------|
| Capacidade | 245 Wh |
| Potência nominal | 300W |
| X-Boost | até 600W |
| Recarga AC | 60 min (0-100%) |
| Bateria | LiFePO₄ — 3.000 ciclos |
| Peso | ~3,5 kg |
| Dimensões | 24,5 × 21,4 × 14,2 cm |
| Saídas | 1x AC, 2x USB-A, 1x USB-C 100W, 1x Veicular |
| Entrada solar máx. | 110W |
| Ruído | <30 dB |
| UPS | <20 ms |

### Diferenciais
- 2x mais autonomia para aparelhos <100W
- 30% mais compacto que modelos anteriores
- Silencioso como biblioteca
- 4 opções de entrada
- Alimenta 6 dispositivos simultaneamente
